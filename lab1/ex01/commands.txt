touch <имя.файла> # создание пустого файла с возможностями изменения метаданных

df # сколько места свободно на диске

ll # Команда для просмотра содержимого директорий. Более современная альтернатива этой утилиты ls -l может работать не во всех дистрибутивах Linux.

cat # (Concatenate / Catenate). Команда двойного назначения. Показывает на экране что содержит файл или стандартный ввод. Также к «склеивает» несколько переданных подряд файлов в один. Если нужно посмотреть содержимое одного файла, вводим: $ cat имя_файла

cp # (Copy). Утилита, позволяющая скопировать файл или каталог. Копирование файлов и каталогов. Чтобы копирование прошло рекурсивно, т. е. включало все поддиректории и файлы в них, надо добавить к команде: $ cp -r

mv # (Move). Эта команда в Linux одновременно отвечает за перемещение и переименование файла или каталога. При переименовании файл перемещается в ту же папку, но уже названную по-новому.

ln # (Link). Создание жестких и символических ссылок. 
# Жесткие ссылки создают непосредственную связь, поэтому индексовый дескриптор ссылки и файла будут одиннаковые, можно перемещать и ссылка будет рабочей, нельзя ссылаться на каталог. 
# Символические - можно дать файлу другое имя, но не связывать файл с индексным дескриптором, при удалении/перемещении/переименования файла ссылка остается, но не работает, можно ссылаться на каталог. 

clear # Команда полной очистки окна терминала удаляет все ранее выведенные сообщения.

history # Показывает ранее введённые пользователем команды. Перемещаться по списку введённых команд можно, нажимая на клавиатуре «Вверх» и «Вниз».

man <команда> # открывает мануал для команды
